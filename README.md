# Smart-Cognitive-Driver-Safety-System
This project presents an advanced real-time driver monitoring system designed to improve road safety by detecting drowsiness, distraction, and fatigue using computer vision techniques. Unlike traditional systems that rely on a single parameter, this system integrates multiple behavioral indicators such as Eye Aspect Ratio (EAR), head pose estimation, and blink patterns to assess driver alertness.

The system computes a dynamic Driver Risk Score based on real-time analysis and triggers intelligent alerts when unsafe conditions are detected. It also logs driver behavior data for further analysis, making it both a practical application and a foundation for future research.
